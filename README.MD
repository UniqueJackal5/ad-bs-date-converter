# Nepali Date Converter | AD to BS & BS to AD

A professional, mobile-friendly, and SEO-optimized web tool for converting dates between the English (AD) and Nepali (BS - Bikram Sambat) calendars. 

## 🌐 Live Demo
The web version can be deployed instantly to Netlify or any static hosting service. Simply open `index.html` in your browser to use the tool locally.

## ✨ Key Features
- **AD to BS & BS to AD Conversion**: Precise conversion using historical lookup tables (2000 BS - 2090 BS).
- **Bold Classic Design**: A high-contrast, tactile UI with large fonts for maximum readability.
- **Full Nepali Language Support**: One-click toggle to switch the entire interface and date results to Nepali.
- **SEO Optimized**: Includes meta tags and OpenGraph support to improve visibility on search engines and social media.
- **Mobile Friendly**: Fully responsive design that works seamlessly on all screen sizes.

## 🛠 Project Structure
- `index.html`: The main web application interface.
- `style.css`: The "Bold Classic" styling and responsive layout.
- `converter.js`: Core conversion logic with English and Nepali support.
- `legacy-desktop/`: Original C++ source code and Windows build assets for historical reference.

## 🚀 Deployment to Netlify
1. Connect your GitHub repository to Netlify.
2. Set the **Publish directory** to the root folder (or `.` ).
3. Netlify will automatically serve the `index.html` file.
