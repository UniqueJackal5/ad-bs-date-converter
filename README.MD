## Quick ADBS Date Converter

A lightweight tool to quickly convert AD dates to BS (Nepali) dates and vice versa.

<p align="center">
	<img alt="Quick ADBS Date Converter Logo" src="build/adbslogo.png" width="200" height="200"/>
</p>

### Key Features
- **New: Web Version** – Run the converter directly in your browser using the TypeScript/JS port.
- **Desktop Application** – Original Windows C++ application (approx. 1 MB).
- **Fast & Optimized** – Uses a high-performance lookup table for accurate conversions.

---

### 🌐 Web Demo
To try the web version immediately:
1. Navigate to the `typescript/` folder.
2. Open `index.html` in any web browser.
3. Use the interactive interface to convert dates instantly.

### 💻 Technical Info
The core logic has been ported from the original C++ implementation to **TypeScript/JavaScript**, making it easy to integrate into modern web projects.
